What We Do

The CVLore service provides an interactive visual workspace that simplifies the creation, testing, and execution of a vision algorithm model. You put the data and the supplied modules on an interactive canvas and link them together to create an algorithm that is then executed. No programming is required, only a visual connection of data sets and modules is performed in order to create a model of a technical vision algorithm.

The CVLore service will be used for educational purposes to train new members of the RobotLab student association in the basics of technical vision tasks and algorithms. All RobotLab members will have access to it. In this service, a demonstration will be made on the construction of scenarios for working with technical visions. It is assumed that this service can be used for prototyping solutions to various technical vision problems and testing hypotheses.

OpenCV (Open source computer vision)

OpenCV is a library of programming functions mainly aimed at real-time computer vision. Originally developed by Intel, it was later supported by Willow Garage then Itseez (which was later acquired by Intel). The library is cross-platform and free for use under the open-source BSD license. OpenCV supports the deep learning frameworks TensorFlow, Torch/PyTorch and Caffe.
